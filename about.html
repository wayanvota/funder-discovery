About Funder Discovery

Find funders actually worth an NGO leader's time.

Funder Discovery is a public prototype for building a defensible funder shortlist from NGO context, public funder data, and 990-backed evidence. The ranking is deterministic so the logic can be audited.

Who can use it

The intended product should support US 501(c)(3) nonprofits working in the US and US 501(c)(3) nonprofits working overseas. The current public prototype accepts overseas-work inputs, but it is not yet a reliable international funder search because it reranks a small curated seed set.

What it does

The tool asks for enough intake detail to evaluate mission fit, geography, ask size, evidence strength, relationship risk, and eligibility risk. It then ranks funders and produces a selected funder brief with warnings and ask guidance.

What it does not do yet

This static version does not store user submissions, run live 990 ingestion, or call an AI model from the browser. A Render backend can add generated reports later without exposing private API keys.

Data posture

The current funder dataset is curated inside the source code for auditability. A production version should add filing-year provenance, recipient-level grant examples, and last-verified dates.
