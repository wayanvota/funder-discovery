About Funder Discovery

Find funders actually worth an NGO leader's time.

Funder Discovery is a public tool for building a defensible funder shortlist from NGO context, public funder data, and 990-backed evidence. The backend discovers candidates first, then the ranking logic scores fit, warnings, and ask ranges.

Who can use it

The intended product should support US 501(c)(3) nonprofits working in the US and US 501(c)(3) nonprofits working overseas. The current public version should not use a static shortlist. Overseas-work searches still need explicit geography evidence for each funder.

What it does

The tool asks for enough intake detail to evaluate mission fit, geography, ask size, evidence strength, relationship risk, and eligibility risk. It then ranks funders and produces a selected funder brief with warnings and ask guidance.

What it does not do yet

This static frontend does not store user submissions or call an AI model from the browser. It sends discovery requests to the Render backend, which needs an OpenAI API key for live funder discovery.

Data posture

Dynamic discovery should use foundation websites, ProPublica, IRS and 990 references, Kindora pages when available, and public grant guidelines. A production version should add filing-year provenance, recipient-level grant examples, and last-verified dates.
